# rogue-descent

Real-time twin-stick isometric roguelike dungeon crawler — TypeScript + Three.js + Vite. Part of OleyArcade.

## Local development

```bash
npm install    # install dependencies (Node 24.x — see .nvmrc)
npm run dev    # start the Vite dev server
npm run build  # type-check (tsc) + production build
npm test       # run the Vitest suite (pure game layer)
```
